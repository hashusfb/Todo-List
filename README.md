# Todo-List

Web-based, simple todo list application.

# Requirements

+ PHP 7
+ Mysql

# Technologies

## Back-End
+ PHP
+ Codeigniter 3.1.8
+ Mysql

## Front-End
+ HTML
+ CSS
+ JavaScript
+ Jquery v3.2.1
+ Bootstrap v3.3.7
+ MomentJS v2.18.1
+ Datepicker Bootstrap v1.7.1
+ Switchery v0.8.1
+ DateRangePicker v3.0.3
